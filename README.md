# End-to-End Shopify Sales & Customer Analytics Dashboard | Power BI

## Project Overview

This project analyzes Shopify sales performance, customer purchasing behavior, customer retention, and product trends using Power BI.

## Tools Used

-Power BI
-Power Query
-DAX
-Excel
-Data Modeling

## Project Objective

To analyze Shopify sales and customer behavior, identify revenue-driving products and regions, and measure customer retention performance through interactive Power BI dashboards.

## Business Recommendations

* Increase marketing investment in California and Texas, as they generate the highest revenue.
* Promote Shopify Payments through checkout incentives because it contributes the majority of sales.
*  Launch retention campaigns targeting one-time buyers to improve the 46% repeat purchase rate.
*  Focus inventory planning on top-performing categories such as Running Shoes and Tennis Shoes.


## Key KPIs

* Net Sales: $4.18M
* Total Orders: 7,534
* Total Customers: 4,431
* Repeat Customers: 2,039
* Repeat Rate: 46%
* Average Order Value: $562.6

## Dashboard Features

* Sales Performance Analysis
* Customer Behavior Analysis
* Customer Retention Metrics
* Product Performance Analysis
* Geographic Sales Analysis
* Payment Gateway Analysis

## Business Questions Answered

* Which states generate the highest revenue?
* Which product categories contribute most to sales?
* What percentage of customers are repeat buyers?
* Which payment methods drive the most revenue?
* How does sales performance change over time?

## Key Insights

* California generated the highest revenue among all states.
* Shopify Payments contributed the majority of total sales.
* Repeat customer rate reached 46%.
* Running Shoes were among the highest revenue-generating product categories.
* Sales showed strong performance during peak seasonal periods.
